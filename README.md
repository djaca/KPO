# KPO knjiga
Web aplikacija za lako popunjavanje kpo knjige i eksportovanje u pdf.

## Instalacija
```
git clone git@github.com:djaca/kpo-obrazac.git
cd kpo-obrazac
composer install
yarn
cp .env.example .env
php artisan key:generate
Add database credentials in .env file.
```
